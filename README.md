# Uptobox API

## Requirement

Retrieve your API token from your Uptobox account : [How to find my API token ?](https://docs.uptobox.com/#how-to-find-my-api-token)

## Install

`npm i uptobox-api`

## Usage

`const uptobox = require('uptobox-api');`

## Examples
